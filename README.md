JDBCloner
=========

Clone rows, tables or whole databases. (Schemas, relations and information)
beetween diferent or same RDBMSs.

<h2>Powered by</h2>
<ul>
  <li>Java jdbc</li>
  <li>spring-jdbc</li>
</ul>

<h2>Supported databases (More incomming)</h2>
<ul>
  <li>Oracle</li>
  <li>MySQL</li>
</ul>
